Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS.

Set up Visual Studio Code 1.Download and install VS Code(Visual Studio Code). 2.Create a new file. 3.See an overview of the user interface. 4.Install support for your favorite programming language. 5.Change your keyboard shortcuts and easily migrate from other editors using keymap extensions. 6.Customize your editor with themes.

An instance of a Windows Console has a screen buffer and an input buffer. A console is the combination of a monitor and keyboard. Here is how to open console screen in 3 easy steps: 1.Right click on the white space. 2.click on Inspect button. 3.click on console which is next to Elements button. You can print anything in the console screen using this code:- console.log
